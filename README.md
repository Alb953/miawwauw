# Adopta Miauw Wau — Codex Documentation Package

This folder contains the full build context for Codex.

## Start here

Open `00_START_HERE.md`, then follow `04_FRONTEND_STEP_BY_STEP_NEXT.md` first and `06_BACKEND_STEP_BY_STEP_EXPRESS.md` second.

## Most important rules

- Build the frontend first.
- Use **Next.js**, TypeScript, App Router, and Tailwind CSS.
- Do **not** use React Router.
- All visible frontend text must be in **Spanish**.
- Use `assets/template.png` as the visual style reference.
- Build the backend after the frontend with Node.js + Express.
- Use Supabase PostgreSQL and Cloudinary through the backend only.
- Keep Codex context updated in `10_BUILD_STATE_CHANGELOG.md`.
